Template de site para adoção de cachorros

Desenvolvi este template para reforçar meus estudos de programação com as linguagens HTML, CSS, React e Nodejs. No site existe uma página inicial, onde apresenta textos e funcionalidades de navegação entre links com React Router. Existe uma página de adoção onde o usuário preenche um formulário dinâmico e o envia. Assim que o formulário é enviado ele gera um e-mail com o nodemailer e joga o usuário para uma tela de "Sucesso". Existe também a página "Sobre" onde eu explico sobre a ideia do template. Ele foi totalmente inspirado no site do acaochego.org
